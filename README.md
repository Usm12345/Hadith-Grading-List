# Hadith Grading Lists

## Overview

**Hadith Grading Lists** is an open-source repository containing numbered lists of graded Ahadith compiled from major Hadith collections.

The purpose of this project is to provide researchers, students of knowledge, scholars, developers, and AI practitioners with structured Hadith grading datasets that can be easily referenced, filtered, and integrated into software applications, research projects, presentations, and educational resources.

This repository was created during the development of **A.T.O.M**, an Islamic RAG-based AI chatbot. During development, it became apparent that obtaining structured and numbered grading lists was a time-consuming process. Although grading information existed across multiple sources, concise and digitized lists suitable for software and AI applications were difficult to find.

This repository aims to bridge that gap by providing organized grading datasets in a simple and accessible format.

---

# Included Hadith Grades

The repository currently contains the following Hadith grades:

### Da'if (ضعيف) — Weak

* Weakness in the chain or narrators
* Insufficient reliability for independent legal rulings

### Mawdu' (موضوع) — Fabricated

* Falsely attributed to the Prophet ﷺ

### Munkar (منكر) — Denounced

* A weak narrator contradicts trustworthy narrators

### Shadh (شاذ) — Irregular

* A reliable narrator contradicts narrators who are more reliable

# Numbering System

The grading lists have been compiled using:

* International Hadith Numbering
* English Translation Numbering

wherever applicable.

This allows easier cross-referencing between:

* Printed editions
* Digital databases
* Research papers
* Academic projects
* AI and Machine Learning applications
* Islamic software development

# Disclaimer

This repository is a compilation project intended for educational, research, and software-development purposes.

While every effort has been made to ensure accuracy, errors may still exist. Users are encouraged to independently verify Hadith gradings through recognized scholarly sources before using them in academic, legal, or religious rulings.

The compiler does not claim independent authority in Hadith authentication. All grading information is derived from established scholarly works and published references.
